# Money Maker (C#)

This C# program converts a given amount of cents into gold, silver, and bronze coins using the fewest coins possible. It validates user input, uses integer arithmetic for accurate calculations, and displays results with proper singular/plural wording.

**Features:**

* Validates input to ensure a non-negative integer
* Converts cents into gold coins (10), silver coins (5), and bronze coins (1)
* Properly displays singular/plural for coins and cents
* Demonstrates loops, integer arithmetic, and conditional operators

**How to Run:**

1. Save the code as `Program.cs`.
2. In a terminal, create a console project if needed:

   ```bash
   dotnet new console -o MoneyMakerApp
   cd MoneyMakerApp
   ```
3. Replace the generated `Program.cs` with your code.
4. Run:

   ```bash
   dotnet run
   ```
